# PackDev

PackDev is a gradle plugin for creating and building modpacks. It supports both [CurseForge](https://www.curseforge.com/minecraft) and [Modrinth](https://modrinth.com/) and can build modpacks fo [Forge](https://files.minecraftforge.net), [Fabric](https://fabricmc.net/) and [Quilt](https://quiltmc.org/).

Best used with [Moonstone](https://github.com/ModdingX/Moonstone).

For more information, see the [Wiki](https://moddingx.org/wiki/packdev).
